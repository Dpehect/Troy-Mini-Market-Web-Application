# Portfolio Presentation Notes

## Short description

Troy Mini Market is a professional grocery ecommerce experience designed around minimum user effort. It combines a premium ultra-minimal visual system with real ecommerce functionality: product search, filters, product pages, cart, checkout, favorites, reorder and order success flows.

## Interview explanation

I wanted the project to feel like a real market storefront, not a student demo. The design direction is intentionally restrained: large typography, clear product hierarchy, strong photography, very few homepage sections and fast actions above the fold.

The core UX idea is simple: grocery customers usually know what they need. So the product starts with search, one-tap essentials, dinner kit and deli pickup instead of forcing users through a decorative landing page.

The process page is separated from the storefront so the customer experience stays commercial while the portfolio review still has design and engineering context.

## Technical highlights

- Next.js App Router
- TypeScript typed product system
- Zustand persisted cart/favorites/orders
- React Hook Form + Zod checkout validation
- Query-aware search page
- Dynamic product and category routes
