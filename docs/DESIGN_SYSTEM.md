# Design System Notes

## Direction

The visual language is ultra-minimal and product-focused:

- warm neutral background
- dark foreground contrast
- restrained orange accent
- large editorial typography
- generous spacing
- clear product hierarchy
- minimal borders and soft surfaces

## Homepage hierarchy

The homepage is reduced to:

1. hero image
2. one primary search entry
3. direct cart actions
4. minimal shortcut cards
5. fast carts
6. product preview

This prevents portfolio-style over-decoration from interfering with actual shopping.

## Product card hierarchy

Every card keeps the same priority:

1. image
2. category
3. product name
4. brand and size
5. price
6. stock/delivery cue
7. add-to-cart action

## Motion

Motion is restrained:

- short entrance transitions
- product card hover
- feedback-oriented interactions

Motion is not used to delay shopping.
